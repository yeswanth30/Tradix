Tradix API Documentation

1. Stock Data APIs:

  • /getstockdata - (GET) Gives you All Stocks and their corresponding DATA (Id, Name, Price, Time, Preprice, Updated_time)
  • /givestock/id - (GET) Gives You Stock that the user Clicked #You have to send Stock_id

2. User Credential APIs:

  • /signin - (POST) Requires Email, Password
  • /login - (POST) Requires Email and Password #Gives you all the DATA of the particular User

3. Stocks and Users APIs:

  • /buyit - (POST) Requires User_id, Stocks_id, Stock_price, Count
  • /sellit - (POST) Requires User_id, Stocks_id, Stock_price, Count
  • /getbought/id - (GET) Requires User_id #Gives you all the stock data of the User who Bought
  • /getsold/id - (GET) Requires User_id #Gives you all the stock data of the User who Sold
  • /updatebalance - (POST) Requires Balance value to update user's Balance

4. News APIs:

  • /allnews - (GET) Gives You all News DATA
  • /getnews/id - (GET) Gives you all the news DATA of a particular news that the user pressed #Requires Id of news
